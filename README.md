# 👑 KING OF FUTURES INDONESIA (KOFI)
## Free Trading Dashboard for Binance Futures

### Features
- ✅ Real-time BTC price from Binance API
- ✅ 10 Coin prices live
- ✅ Fear & Greed Index
- ✅ Funding Rate
- ✅ Technical indicators (RSI, MACD, BB, ATR)
- ✅ Trading signals (LONG/SHORT)
- ✅ 7-day predictions
- ✅ Trading journal (localStorage)
- ✅ Dark mode
- ✅ Responsive design

### Deployment (FREE)
1. Create GitHub repository
2. Upload all files
3. Settings → Pages → Deploy from main branch
4. Your app live at: https://username.github.io/repo-name

### Requirements
- Modern browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for API data)

### API Used (All FREE)
- Binance Futures API (no key needed)
- Alternative.me FNG (no key needed)

### Security
- All data stored locally (localStorage)
- No backend, no database
- API keys NOT stored anywhere
